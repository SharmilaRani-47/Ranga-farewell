## 🚀 How to Customize

### 1. **Change Images**
Edit `index.html` and replace image URLs in the gallery section:
```html
<img src="YOUR_IMAGE_URL" alt="Description">
```

### 2. **Update Videos**
Replace YouTube video IDs:
```html
<iframe src="https://www.youtube.com/embed/YOUR_VIDEO_ID"></iframe>
```

### 3. **Modify Text Content**
Edit all text, headings, and descriptions directly in `index.html`

### 4. **Change Colors**
Update the color variables in `styles.css`:
```css
:root {
    --primary-color: #8b5cf6;      /* Purple */
    --secondary-color: #ec4899;    /* Pink */
}
```

### 5. **Add Social Links**
Update social media URLs in the Contact section:
```html
<a href="https://www.facebook.com/YOUR_PAGE">Facebook</a>
```

## 🔧 Troubleshooting

**Site not showing?**
- Wait 1-2 minutes for GitHub Pages to deploy
- Refresh the page (Ctrl+Shift+R for hard refresh)

**Videos not playing?**
- Check video URL is accessible
- Ensure MP4 format for local videos
- YouTube videos work with any public video ID

**Images not showing?**
- Verify image URLs are correct
- Check image is publicly accessible
- Try different image hosting service
